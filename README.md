API

init()	инициализирует канал и запускает принятие сообщений.

send(message:any) message:сообщение	отправляет сообщение всем вкладкам, подписанным на этот канал.

onMessage(handler:function): function:функция которая должна выполнятся при получении сообщения. подписывает вкладку на канал с сообщениями возвращает функцию для отписки от канала.

destroy()	закрывает канал и удаляет всех пользователь.
