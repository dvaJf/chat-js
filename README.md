API

init()	инициализирует канал и запускает принятие сообщений.

send(message:any) message:сообщение	отправляет сообщение всем вкладкам, подписанным на этот канал.

onMessage(handler:function): function:функция, которую можно передать в offMessage для отписки. подписывает вкладку на канал с сообщениями возвращает функцию для отписки от канала.

offMessage(handler: function) Отписывает от канала.

destroy()	закрывает канал и удаляет всех пользователь.
